关于世界杯的记忆

世界杯，很多时候专门指的是足球世界杯。对于只是一个伪球迷的我，谈谈记忆里的世界杯。 94年以前的世界杯，我的印象是一片空白。因为在农村长大，村子里一直到92年才通上电，没电就没有电视，没有电视也就是没所谓的足球了，更何况你让一个在94年之前都没有见过足球的人去喜欢足球，那更是一件不怎么可能发生的事情。所以，1994年的世界杯，什么马拉多纳吸毒什么什么的当时根本就不知道。94年之前对于足球的记忆，是在某个夏天的夜晚，我和哥哥躺在院子里，他拿了一个收音机在听一场足球赛，应该是中国队和别的什么国家踢，在宋世雄老师解说的声音中，我进入了梦想。 第一次真正意义的关注世界杯，应该是在1997年，当时我高四，同桌很喜欢踢球，中国队踢10强赛，忘记那一场了，大概率是对沙特，周六的下午，我们还在上课，他拿了一个收音机，听完了一场比赛，结果是中国队输了，气愤的不得了，班里的几个女生也喜欢足球，课余的时候还经常聊这个，这算是了解足球的开始。 98年的世界杯，在gogogo的音乐中，我领到了我的大学录取通知书。宿舍里有个体育迷，经常买《体坛周报》，没事了就拿过来看，知道了杨晨，谢辉。还曾和那个同学一起晚上去录像厅看德甲。2001年，在郑州大学的一个食堂里，和里面的几百个同学一起见证了中国队的出线。尤其记得张恩华的那个头球，因为在罚这个角球的时候，我给我那个同学说，我有预感这个球会进，结果，球真的进了。 2002年的世界杯，对于我们来说正值毕业季，由于有中国队的参加，班里几个喜欢看球的同学专门租了一间房子来看球，我也就经常去蹭。决赛的时候，学校已经开始让我们离校了，记得是在一个有女朋友的同学家，带了些啤酒和凉菜，并吃边喝边看球。 在后来，踏入了社会，往往为了生活而奔波，也没有了看球的闲情逸致，也不是没有看，只不过都已经印象不深了，没有什么好记录的。

Django设置使用MySql的时候遇到的一些坑

Django默认使用的数据库是sqllit,如果是一个小项目的话是完全够用的，但是如果考虑到这个项目未来的话，一般都会换成Mysql或者其他更专业的数据库。本文，就来说说，在设置使用mysql的时候，遇到的一些问题及解决的方法。 项目里设置引用Mysql 在setting.py里,把原来使用sqllit的语句删除或者注释掉，添加如下代码 DATABASES = { 'default': { 'ENGINE': 'django.db.backends.mysql', # 数据库引擎 'NAME': '', # 数据库名，先前创建的 'USER': '', # 用户名，可以自己创建用户 'PASSWORD': '', # 密码 'HOST': '127.0.0.1', # mysql服务所在的主机ip 'PORT': '3306', # mysql服务端口 } } 安装Python访问MySQL的模块 Django官方已经不建议使用pymysql库了，而是改用mysqlclient，直接pip安装即可。 pip install mysqlclient 在debian11上安装的时候，显示OSError: mysql_config not found的错误，查询得知，需要安装libmysqlclient-dev，但是用apt install libmysqlclient-dev的时候，安装不成功，提示是另一个分支，建议安装libmariadb-dev。所以安装这个即可。 apt install libmariadb-dev 出错：error: command ‘x86_64-linux-gnu-gcc’ failed: No such file or directory sudo apt-get install build-essential python3-dev libssl-dev libffi-dev libxml2 libxml2-dev libxslt1-dev zlib1g-dev

这波疫情应该是到了尾声了

从10月11日开始，到13日正式静默，到今天大部分小区解封，这波疫情算是到了尾声了。这20多天里发生的很多的事情，在自媒体发达的今天，事情的真假愈发扑朔迷离，而最好的对待的方法也就是静而观之，看看就行，心里一定要有自己的主见，不然很容易就被带了节奏。至于，为什么从五一的模范生到今天这个地步，老刘认为，主要是以下方面。 信息的不透明 这次的疫情从开始就透着诡异，事前就没有听到什么风声，老刘预料到了国庆过后，会有这么一波，但是没有料到会是这么严重。更离谱的是官方不再发声，以前的时候，哪里有确诊，密接的情况，流调的结果等都会很快的公布出来，人们看到这些就会心里有数，这次的现实是，官方没有了通报，如果你不是被关在了家里，如果你不是看到身边的朋友一个一个拉走或隔离或治疗，你都觉得根本就没有疫情的发生。 至于富士康的事情，如果和之前的处理方式相同的话，这件事情根本不会发生。富士康是有它做的不到位的地方，根本的原因还是信息，没有真信息，就会被谣言所包围，十几二十岁的年轻人，处于那种环境，生活物质不足，病人隔离不及时，核算检测不及时，加上对这个病毒的恐怖，很容易造成心态的崩溃，发生这样的事情也就不奇怪了。 病毒的感染力太强 尽管都是轻症，但是不可否认的是，感染力真的很强，问过确诊的朋友，有症状的时候确实很难受。加上国庆期间人员流动太频发，造成了一发不可收拾的地步，显然，官方也没有料到会这么严重，之前准备的定点医院，隔离酒店等明显不足，雪上加霜的是，刚刚接受过兄弟城市的一波确诊，让本来不足的情况更加捉襟见肘。 老方法不一定适合新问题 老子说，“重为轻根，静为躁君”。你认为已经把握住了事情的要点，将来出现问题的往往也是你认为重点的地方。纵观整个历史，都逃脱不了这个范畴。封控，在之前之所以能起到良好的效果，是因为占据着天时地利和人和，但是现在病毒在变异，经过多轮静默与封控，人也已经疲惫不堪，是时候到了调整一下让整个城市暂停的策略了。

白牦牛肉炖萝卜

前几天，老田大爷的儿子给快递过来一些白牦牛肉，正宗的甘肃武威天祝县的白牦牛。他媳妇娘家自己弄的。挑了最好的排骨部分给发过来的。 要说牛肉不稀罕，豫东黄牛肉也是比较出名的，不过现在都买不到正宗的了，去菜市场买过几次牛肉自己卤，味道也还行，只不过还是感觉没有小时候的牛肉好吃，也不知道是小时候实在嘴馋还是现在肉真的变差了。 说的有点远了，回到今天我们说的主题，白牦牛肉！白牦牛算是天祝县的特产了，生活的高原地带，肉质细腻，营养丰富。舌尖上的中国里说，高端的食材往往只需要最朴素的烹饪方式。这样才能带来食材本身的味道。 把牛肉用清水洗净，凉水下锅，放点料酒，加两片生姜，大火烧开，煮出血末等杂质。捞出牛肉，清水冲洗干净。 砂锅里添水，放入姜片，可以多放一些，放点料酒。牛肉也放进去。大火烧开后，转中火炖，一直炖，一直炖，差不多要炖3个小时，所以，水要添足。3个小时后，放入萝卜，盐，再炖差不多半个小时，就可以出锅了。 至于味道吗，真的是太香了。
