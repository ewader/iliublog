佩戴角膜塑形镜可引起角膜的这些变化

角膜塑形镜是目前非常流行的一种控制近视的一种手段，从研究的效果来看是目前有效率最高的一种。但是现在各种媒体、医院、各网红眼科医生们却大都默默的忽略了佩戴隐形眼镜的风险。今天，老刘在看国外的文献的时候，发现了如下的一篇文章，翻译过来给大家，以便大家权衡利弊。 该研究包括选择了32位白人近视眼患者，他们均佩戴了三个月的角膜塑形。使用光学相干断层扫描（OCT）血管造影对患者治疗前后的总、上皮和基质厚度，以及沿八个半圆周的中心、旁中心和外周值进行测量。 研究发现，佩戴ortho-K后，平均中央角膜总厚度比佩戴前薄4.7µm±1.0µm。旁中心角膜厚度没有变化，但中周边角膜厚度增加了3.3µm ±1.6µm，这主要是上皮的影响。在所有样本中，镜片的适配偏心率小于1.0mm，其中大多数表现为水平-颞侧型（87.5%）。角膜地形图分析显示，水平和垂直上皮厚度不对称变化的模式具有旁中心颞侧最薄的数值，而中周边鼻侧数值最厚。 角膜地形图和上皮厚度在基线（左）和三个月ortho-K治疗期后（右）之间的变化。注意中央和颞部上皮变薄，中周部增厚，与本研究中观察到的颞部偏心现象明显相关（图来自本研究，根据知识共享4.0使用）。照片：González-Pérez J等人。J Optom。2023；17（2）。点击图片放大。 另外，一些研究发现有基质增厚的情况，而另一些研究则没有。这项研究只发现旁中心区域有基质增厚的趋势（+3.9%）。 这项研究还观察了水平和垂直子午线上上皮厚度的变化。水平差异与另一项最近的研究相似，但垂直差异是只在上皮优于上方的区域中描述的新情况。 最后，观察到镜片偏心倾向于向颞下移动，这也与以前的研究一致。垂直镜片偏心与区域上皮或基质变化无关；然而，佩戴ortho-K后上皮变化与水平镜片偏心程度有关。水平偏心程度也与中周部上皮变化（鼻侧和上方）呈正相关，与中周部颞侧上皮变化呈负相关。 在临床方面，作者们确认，“对于正在进行ortho-K的患者，为了更好地监测角膜组织的重新分布，最好能有一种仪器可以自动进行更大范围的分析，至少覆盖角膜中央8毫米的区域，这是目前大多数AS-OCT系统无法实现的。” 本文原文： Mid-peripheral Corneal Thickening Linked with Ortho-K Use 总而言之，研究表明的是，佩戴角膜塑形镜三个月后，出现的情况是：角膜中心变薄，周边变厚。目前没有看到停止佩戴角膜塑形镜后，会不会恢复的报道。但这也是有风险的。如果持续佩戴，角膜中心进一步变薄，周边变厚的话，圆锥角膜的风险就不得不考虑了。

关于近视防控的话题

最近很多家长问老刘： 孩子近视了，戴蔡司成长乐有没有效果？ 孩子近视了，戴康耐特全焦点有用么？ 孩子近视了，戴豪雅新乐学有作用么？ 孩子近视了，戴OK镜怎么样? 孩子近视了，滴阿托品能不能止住度数？ 孩子近视了，是不是晒晒太阳就行了？ 。。。。。。 诚然，孩子近视了，家长着急是一定的，这个世界上有因就有果。我想问问各位家长，孩子眼睛是一开始就近视的吗？既然一开始都不近视，为啥后来近视了？ 孩子平常的用眼习惯好不好？手机等数码产品用的多不多？坐姿正确不？握笔姿势怎么样？ 我就问这四个问题，其他的课外班多不多，作业写到几点，一天能睡多久，有没有户外活动的时间，这些我都不问了，你就回答我上面的四个问题。 就如身体的其他疾病一样，都是一些不良的习惯造成的，你不改变生活习惯，光靠吃药，这个病能好吗？我一个朋友，好喝酒，每次不尽兴都不行，血压高的降压药都很难控制。现在，知道厉害了，减少应酬，每天跑步锻炼身体，血压也慢慢的正常了。 眼睛近视也是一样的道理，近视它是病。治病不仅仅要有药，还得改变习惯，近视防控它是一个系统工程，光指望晒晒太阳，滴滴阿托品或者戴个OK镜，不从根本上找原因，效果能多理想那还真是奇了怪了。 所以，在这里，我奉劝家长们，从根本上重视起来，如果能把近视防控当成语文数学英语那样，每天必做，家长配合，孩子认真，就能很好的控制近视的发展，眼轴的生理增长需要控制，视疲劳及时的消除。 家长们，让我来监督你们的近视防控完成的情况吧，需要的加我微信，我将每日监督你们完成的情况。
